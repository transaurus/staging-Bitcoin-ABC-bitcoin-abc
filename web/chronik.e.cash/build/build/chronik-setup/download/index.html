Download & Install Chronik

Download Chronik for the right platform.

warning

This tutorial is about how to run a bitcoind instance with Chronik enabled. This requires a decent server and may take days to sync. Recommended for advanced users. You can already get started by using one of our pre-synced instances:

  • If you want to know how to use chronik-client in your eCash apps, go to Chronik - JS/TS Client.

Download the right Bitcoin ABC Chronik build for your operating system:

Linux​

Linux x86_64​

Click here to download Bitcoin ABC Chronik for Linux x86_64

 
curl -O https://download.bitcoinabc.org/0.0.0/linux/bitcoin-abc-0.0.0-x86_64-linux-gnu.tar.gz
tar xf bitcoin-abc-0.0.0-x86_64-linux-gnu.tar.gz
rm bitcoin-abc-0.0.0-x86_64-linux-gnu.tar.gz

Linux Aarch64​

Click here to download Bitcoin ABC Chronik for Linux aarch64

 
curl -O https://download.bitcoinabc.org/0.0.0/linux/bitcoin-abc-0.0.0-aarch64-linux-gnu.tar.gz
tar xf bitcoin-abc-0.0.0-aarch64-linux-gnu.tar.gz
rm bitcoin-abc-0.0.0-aarch64-linux-gnu.tar.gz

Windows​

Click here to download Bitcoin ABC Chronik for Windows

 
curl -O https://download.bitcoinabc.org/0.0.0/win/bitcoin-abc-0.0.0-win64-setup-unsigned.exe
tar xf bitcoin-abc-0.0.0-win64-setup-unsigned.exe
rm bitcoin-abc-0.0.0-win64-setup-unsigned.exe

MacOS​

Click here to download Bitcoin ABC Chronik for MacOS x86_64

 
curl -O https://download.bitcoinabc.org/0.0.0/osx/bitcoin-abc-0.0.0-x86_64-apple-darwin.tar.gz
tar xf bitcoin-abc-0.0.0-x86_64-apple-darwin.tar.gz
rm bitcoin-abc-0.0.0-x86_64-apple-darwin.tar.gz
